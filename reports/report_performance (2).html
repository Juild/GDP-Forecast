def clean_and_pivote(df):
    countries_gdp = df[df["IndicatorCode"] == config.GDP_GROWTH]["CountryCode"].to_list() #countries with gdp
    countries = df["CountryCode"].unique()
    countries_no_gdp = list(set(sorted(countries)) - set(sorted(countries_gdp))) # set difference =  countries with no gdp

    return df.pivot(index=["CountryCode","Year"], columns="IndicatorCode", values="Value").drop(countries_no_gdp)

model = xgb.XGBRegressor(max_depth=9, learning_rate=0.01, subsample=0.8, n_estimators=1000, base_score=y_train.mean())

The proportion of the variance of GDP Growth of the next year that the model can explain is a 32.9\% (R2), which is not much (usually a good model attains at least a 60%).

The closer the values are to the red line, the better the predictions are.

We can see that our model does not fit well the outliers, namely, the points that are more distant to the mean. One possible reason is the imputation method we have used, due to the number of missing values and the distribution of the GDP Worth we decided to impute with the mean. This could make the model less susceptible to predict very high or very low values.
